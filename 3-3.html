设置元素的属性

功能描述

在页面中，创建一个<img>标记，通过jQuery中的attr()方法，在页面加载时，设置该标记的img和title属性值，并显示在页面中。

实现代码

$(function(){
  $("img").attr("src", "http://davidwalsh.name/demo/codepenLogo.png");
  // 设置src属性
  $("img").attr("title", "这是Codepen的Logo");
  // 设置title属性
  $("img").attr({src: "http://davidwalsh.name/demo/codepenLogo.png", title: "这是Codepen的Logo的Logo"});
  // 同时设置两个属性
  $("img").addClass("clsImg");
  // 增加样式
  $("span").html("加载完毕");
  // 显示加载状态
})

代码说明

在页面中，attr()方法不仅可以获取元素的属性值，还可以设置元素的属性，其设置属性语法格式如下表示：attr(key, value)。其中，参数key表示属性的名称，value表示属性的值。如果要设置多个属性，也可以通过attr()方法实现，其语法格式如下所示：attr({key0: value0, key1: value1})。
