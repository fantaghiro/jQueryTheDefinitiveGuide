元素样式操作——直接设置元素样式值

功能描述

在页面中，创建一个p标记，单击该元素时，其中的文字加粗、斜体及增加背景色。

实现代码

$(function(){
  $("p:eq(1)").mouseover(function(){
    $(this).css("cursor", "pointer");
    // 鼠标位于索引号为1的p上时，变为手形
  })
  $("p:eq(1)").click(function(){
    $(this).css("font-weight", "bold");
    // 字体加粗
    $(this).css("font-style", "italic");
    // 斜体
    $(this).css("background-color", "#eee");
    // 增加背景色
  })
})
